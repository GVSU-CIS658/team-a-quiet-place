### 2. Technology stack
- **Vue Router**: Handles navigation between pages
    - URLs
    - History
    - Navigation
    - Route guards

- **Pinia** is global state management: Stores data that multiple components need

- **Vuetify** is a UI component library

- **Firebase** is a backend platform
    - **Firebase Authentication**: Handles user login system
        - Email / Password
        - Google login
        - Apple login
        - Facebook login
    - **Firestore** is a cloud database: Stores app data

- **Fetch** is the native browser HTTP client: Uses to call APIs

- **Axios**

- **ExpressJS** is a Node.js backend framework: Uses to create APIs
